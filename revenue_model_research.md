# Alpha Trip 수익 모델 조사 결과

## 1. 차량 호출 서비스 수수료 구조

### 주요 플랫폼 수수료
- **Uber**: 승객 요금의 25% 기본 수수료 (드라이버에게 75% 지급)
- **Grab**: 20-25% 커미션
- **Bolt**: 15-20% 커미션 (Uber보다 낮음)
- **Gett**: 12% 커미션 (블랙캡)

### 어필리에이트/파트너 수익
- **Uber Affiliate**: 신규 유저당 $5-$20, 첫 주문당 커미션
- **Uber API Partner**: 개발자 어필리에이트 프로그램 - 유저당 $5
- **Karhoo 어그리게이터**: 택시 플릿 통합 → 마크업 모델 가능

### Alpha Trip 수익 포인트
1. **어그리게이터 마크업**: 도매가(net rate)에 15-25% 마크업
2. **어필리에이트 커미션**: 신규 유저 유입 시 $5-$20/건
3. **USDT 환전 스프레드**: 현지통화 → USDT 전환 시 1.5-3% 마진
4. **VAT 차익**: 현지 부가세(7-25%) 제거 후 USDT 가격 제공

## 2. 음식 배달 서비스 수수료 구조

### 주요 플랫폼 수수료
- **Uber Eats**: 레스토랑에 15-30% 커미션 (플랜별)
- **DoorDash**: 15-30% 커미션
- **GrabFood**: 25-30% 커미션
- **Deliveroo**: 25-35% 커미션

### 어필리에이트/파트너 수익
- **Uber Eats Affiliate**: 앱 다운로드당 커미션
- **DoorDash Affiliate**: 첫 주문당 ~$3 (소비자), 가맹점당 최대 $1,000
- **Deliverect 통합**: 30+ 배달 플랫폼 통합 API

### 마크업 구조
- 배달 플랫폼들은 메뉴 가격에 80-92% 마크업 적용 (PostMates 92%, DoorDash 83%)
- 레스토랑 순이익 마진: 3-9% (배달 수수료 후)

## 3. 여행 에이전트 마크업 vs 커미션

### 커미션 모델
- 호텔 판매가 $200 → 10% 커미션 = $20 수익
- 순이익률: 7.5% (결제 수수료 차감 후)

### 마크업 모델
- 호텔 도매가 $150 → 20% 마크업 → $180 판매
- 순이익률: 14.2% (결제 수수료 차감 후)
- **마크업이 커미션 대비 약 2배 수익**

### 하이브리드 접근 (Top Performer)
- 호텔: 마크업 15-25%
- 항공: 커미션 (마진 얇음)
- 투어/액티비티: 마크업 (고마진)
- 트랜스퍼: 마크업
- 럭셔리: 커미션
- 결과: 평균 패키지 마진 18-22%

## 4. USDT 결제 수익 구조

### 크립토 결제 게이트웨이 수수료
- 일반 수수료: 0.5-2% (신용카드 2.9%+$0.30 대비 저렴)
- NOWPayments: 0.5% 수수료
- 환전 스프레드: 1-3% 추가 마진 가능

### USDT 결제 시 수익 포인트
1. **VAT 차익**: 현지 부가세 7-25% 제거 → 사용자에게 할인 제공하면서도 마진 확보
2. **환전 스프레드**: USDT↔현지통화 전환 시 1.5-3% 마진
3. **결제 수수료 절감**: 신용카드 2.9% → 크립토 0.5% = 2.4% 절감
4. **즉시 정산**: 60일 대기 없이 즉시 USDT 수령
